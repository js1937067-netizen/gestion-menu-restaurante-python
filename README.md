# Gestión de precios de un menú de restaurante en Python

## Descripción del proyecto

Proyecto desarrollado para el Problema 2 de la Fase 5 del curso Fundamentos de Programación de la UNAD.

El programa permite gestionar los precios de un menú de restaurante mediante una matriz que almacena productos, categorías y precios base.

Se aplica una promoción del 15% de descuento a los productos que cumplen con la categoría objetivo y el valor mínimo establecido.

## Características

- Uso de matrices.
- Implementación de funciones en Python.
- Uso de condicionales y ciclos.
- Cálculo de precio final con promoción.

## Condiciones de promoción

Categoría objetivo: Plato Fuerte.

Precio mínimo: $20.000.

Descuento aplicado: 15%.
